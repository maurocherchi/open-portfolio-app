## Development

First, run the development server:

```bash
npm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech stack

* Node.js
* Next.js
* Tailwindcss
* Primereact

### Deployment

The project is deployed on GitHub Pages using GitHub Actions.
